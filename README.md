# PokedexV2
Este projeto foi inicialmente criado em HTML, CSS e Javascript puros, porém decidi refazê-lo em React. O intuito é meramente educacional, um projeto pessoal para mostrar que eu consigo extrair e usar dados de API's REST via Fetch e criar componentes, rotas e usar Hooks em React.

# Páginas

## Homepage
A página contém vários Cards com Informações básicas dos pokémons, embora a quantidade de Cards seja propositalmente limitada para evitar perda de desempenho ao realizar o primeiro carregamento da página, é possível usar o botão de "Load More" para carregar mais Cards.

## Information Page
Uma página dedicada a exibir informações sobre o pokémon do Card o qual o usuário interagiu.
